在Windows Server 2012 中使用IE浏览器玩网页游戏或者使用含有flash插件的网页时会出现如下的提示。主要的原因是在windows server 2012中IE 默认是没有安装 Flash Player。下面的步骤为相关的处理方法

步骤阅读

工具/原料

  • WINDOWS SERVER 2012
  • IE

方法/步骤

  1. 1

    启用IE的Flash Player，打开服务器管理器。

    步骤阅读
  2. 2

    选择“管理”-“添加角色和功能”。

    步骤阅读
  3. 3

    打开向导后保持默认，连续点“下一步”，转到“服务器角色”，找到“用户界面和基础结构”，勾选“桌面体验”。

    步骤阅读
  4. 4

    勾选“包括管理工具（如果适用）”-点击“添加功能”

    步骤阅读
  5. 5

    点击“下一步”继续进行安装。

    步骤阅读
  6. 6

    点击“安装”后开始安装。

    步骤阅读
  7. 7

    完成安装后需要重启电脑。

    步骤阅读
  8. 8

    重启电脑后，打开控制面板。可以看到flash插件已经安装完成。现在可以正常使用IE FLASH

    步骤阅读
