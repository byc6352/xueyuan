Android studio项目引入另一个项目作为Libary

 

方法一：使用Android studio 导入lib项目：

注意： lib项目更改处：

在build.gradle 文件下 将

  1. 注释：// apply plugin: 'com.android.application'
  2. 改为：apply plugin: 'com.android.library' 
  3.     
  4. defaultConfig {
  5.         // applicationId "com.example.sdk_lib"  注释这一行
  6.         minSdkVersion 19      //  版本改为和主项目一样
  7.         targetSdkVersion 28   //  版本改为和主项目一样
  8.         
  9.     }

1.主项目：import Module....引入另一个lib项目

2. 将另一个lib项目的地址写入 ，module name 自己填写，finish后，看到主项目内多了一个文件夹my_lib

3. 配置：按照下面步骤，在主项目app的dependencies 点击Module Depend，

4.弹出框，如果第一步导入lib项目成功，这里就会显示，然后勾选，ok后，在上图的app 模块的dependencies 的列表会看到刚才勾选的配置

5. 运行主项目，成功，此方法是一次性导入需要的lib项目包，如果是lib写好的，可以使用

方法二：

如果lib项目需要实时更新到主项目，建议直接用配置来引入：

1. 在主项目 build.gradle 文件下

dependencies {
    implementation project(':my_libs')     // 加入这行，导入包
}
 2. 在主项目 settings.gradle 文件下

include ':app', ':jcejava', ':signal', ':my_libs'

//  引入你本地的lib项目地址，作为导入包

project (':my_libs').projectDir = new File('../sdk-lib/app')

3. 运行主项目，成功，并且lib项目更新代码，主项目中也会随之更新
